# ParentLoginFlow

**Inicio:** tap en "Iniciar sesión" desde Splash o desde ParentRegister.

**Pantallas involucradas:** ParentLogin → ChildProfileSelector (si ya existen perfiles) o ChildCreationWizard (si la cuenta no tiene hijos aún).

**Decisiones:** ¿credenciales válidas?

**Estados:** error de credenciales (⏳ diseño pendiente), loading durante autenticación (⏳ diseño pendiente).

**Resultado esperado:** sesión iniciada, listado de perfiles visible.

**Casos alternativos:** "¿Olvidaste tu contraseña?" → flujo de recuperación (⏳ no documentado, sin capturas en el set actual).
