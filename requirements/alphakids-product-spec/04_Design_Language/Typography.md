# Typography.md

**Familia tipográfica:** sans-serif redondeada y amigable (estilo "Baloo 2" / "Quicksand" / "Nunito" — confirmar fuente exacta contra archivos de diseño; el estilo visual indica terminaciones geométricas redondeadas).

## Escala
| Token | Tamaño | Peso | Uso observado |
|---|---|---|---|
| Display | 32–36px | 800 (ExtraBold) | "¡Bienvenida a AlphaKids!", título de Splash |
| H1 | 26–28px | 700 (Bold) | Títulos de paso del wizard ("Creemos el perfil de tu hijo") |
| H2 | 20–22px | 700 | Nombres de perfil ("Sofía", "Lucas"), subtítulos de sección |
| H3 | 16–18px | 600 (SemiBold) | Títulos de tarjeta ("Mi mascota", "Objetivo del día") |
| Body | 14–15px | 400–500 | Descripciones, textos de ayuda |
| Caption | 12–13px | 400 | Notas legales, "Tu información siempre está protegida" |
| Labels | 13px | 600 | Etiquetas de input ("Correo electrónico", "Contraseña") |
| Button Text | 16px | 700 | Title Case, sin uppercase ("Continuar", "Crear mi cuenta") |

## Color de texto por contexto
- Sobre fondo claro (cards blancas): Deep Navy `#1E2749`
- Sobre fondo de gradiente saturado: blanco `#FFFFFF`, drop-shadow sutil opcional
- Texto secundario: Slate Gray `#5B6478`
- Links: Primary Blue `#3B7DF6`
