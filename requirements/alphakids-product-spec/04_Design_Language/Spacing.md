# Spacing.md

Grid base de 4pt.

- **4** — micro espaciado entre ícono y texto
- **8** — espaciado entre elementos relacionados (chip a chip)
- **16** — padding interno estándar de tarjeta
- **24** — separación entre secciones dentro de una pantalla
- **32** — separación entre bloques mayores (header vs. contenido)
- **48** — margen superior de la mascota hero
- **64** — espaciado entre el contenido y el footer legal de cierre
