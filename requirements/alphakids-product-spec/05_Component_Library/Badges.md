# Badges.md

**Propósito:** distintivos de estado, rango o logro.

**Variantes:**
- Rank Badge — pill con ícono de estrella + texto. Ej: "Explorador Curioso", "Investigador Inteligente".
- Crown Badge — corona dorada sobre avatar (⏳ confirmar significado exacto: ¿perfil destacado/premium?).
- Streak Badge — racha de 7 días, check verde por día.

**Estados:** Default.

**Uso permitido:** máximo 1 rank badge visible por perfil.

**Navegación asociada:** ChildProfileSelector, AdventureHome, ChildCreationWizard.
