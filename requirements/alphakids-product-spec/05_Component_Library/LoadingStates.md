# LoadingStates.md

**Propósito:** comunicar carga de contenido.

**Variantes:**
- Splash Loading — pantalla completa con mascota + mensaje ("Preparando tu aventura educativa… Cargando tu mundo de aprendizaje") + indicador de dots animado.
- Inline Loading — ⏳ pendiente, no observado en botones/formularios.

**Estados:** Default.

**Navegación asociada:** Splash.
