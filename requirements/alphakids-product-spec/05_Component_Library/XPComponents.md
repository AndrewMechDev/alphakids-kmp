# XPComponents.md

**Propósito:** representar experiencia/nivel del niño.

**Variantes:** Level Badge hexagonal (número + label "Nivel X"), XP Progress Bar (fracción "1,250 / 2,000 XP"), XP Pill pequeño en tarjetas de recompensa ("+10 XP").

**Estados:** Default.

**Uso permitido:** el badge hexagonal siempre acompaña al nombre del perfil.

**Navegación asociada:** Header de Home, ChildProfileSelector, RewardCards.
