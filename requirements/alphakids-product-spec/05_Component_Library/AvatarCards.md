# AvatarCards.md

**Propósito:** selección de avatar de personaje para el niño.

**Variantes:** grid 4 columnas x 2 filas, paginación por dots, tabs de categoría ("Animales", "Exploradores", "Fantasía").

**Estados:** Default, Selected (borde azul + check circular), Hover/Pressed (⏳ pendiente).

**Uso permitido:** una sola selección activa a la vez.

**Uso prohibido:** no permitir multi-selección.

**Navegación asociada:** ChildCreationWizard paso 2 (selección de avatar).
