# Headers.md

**Propósito:** encabezado de pantalla con identidad/saludo y accesos rápidos.

**Variantes:**
- Home Header — avatar + saludo ("¡Hola, Lucas!") + rank badge + contadores de monedas/estrellas + notificación.
- Selector Header — avatar genérico de tutor + botón "Configuración".
- Wizard Header — botón atrás + stepper numerado (1–5) + estrella decorativa.

**Estados:** con/sin punto de notificación rojo.

**Uso permitido:** el stepper solo aparece en flujos de wizard de 5 pasos.

**Uso prohibido:** no mostrar stepper fuera de ChildCreationWizard.

**Navegación asociada:** AdventureHome, ChildProfileSelector, ChildCreationWizard.
