# PetCards.md

**Propósito:** presentar mascotas adoptables o la mascota activa del niño.

**Variantes:**
- Selection Card — grid de 3, ícono de rasgo arriba-izquierda, check de selección arriba-derecha, nombre + rasgo principal + tags de personalidad.
- Detail Card — mascota elegida con bio en bullets + input para nombrarla.
- Home Widget Card — mascota activa con nivel y barra de felicidad.

**Estados:** Default, Selected, Locked (⏳ pendiente — relevante para PetUnlocks por nivel).

**Uso permitido:** mostrar siempre nivel/estado de la mascota en el widget de Home.

**Uso prohibido:** no mostrar mascotas bloqueadas como seleccionables.

**Navegación asociada:** ChildCreationWizard paso 3, AdventureHome, PetKingdom/PetMarket (⏳ pendientes).
