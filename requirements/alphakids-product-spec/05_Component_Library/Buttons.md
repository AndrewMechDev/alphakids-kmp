# Buttons.md

**Propósito:** acción primaria/secundaria/enlace en toda la app.

**Variantes:**
- Primary CTA — pill, fondo Primary Blue `#3B7DF6`, texto blanco, ícono opcional a la derecha. Ej: "Continuar", "Crear mi cuenta", "Comenzar mi aventura".
- Text Link — sin fondo, color Primary Blue. Ej: "Iniciar sesión", "Crear cuenta", "¿Olvidaste tu contraseña?".
- Icon Button circular — fondo blanco translúcido, ícono Deep Navy. Ej: flecha "atrás", configuración, notificación.
- Secondary/Outline — ⏳ pendiente de referencia visual, no observado en el set actual.

**Estados:** Default, Pressed (oscurece ~10%), Disabled (`#C9CDD9`, texto blanco 60%), Loading (⏳ pendiente).

**Uso permitido:** una sola acción Primary CTA visible por pantalla/paso.

**Uso prohibido:** no usar Primary CTA para acciones destructivas; no apilar más de 2 Text Links en la misma fila.

**Navegación asociada:** avance de wizard, Login/Registro, confirmación de formularios.
